# @newkrok/nape-pixi

PixiJS v8 renderer integration for [@newkrok/nape-js](https://www.npmjs.com/package/@newkrok/nape-js).

> **Status:** scaffolding only (0.0.0). The runtime API lands in Phase 1 —
> see `ROADMAP.md` in the repo root.

Planned surface:

- `BodySpriteBinding` — keep PIXI sprites in sync with nape bodies
- `FixedStepper` — fixed-step physics with smooth render interpolation
- `PixiDebugDraw` — on-demand shape + constraint + AABB overlay
- `WorkerBridge` — helper for main-thread Pixi + worker-thread physics
